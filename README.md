# Restaurant Tab Manager

This project is a Java application that simulates a restaurant ordering system using object-oriented programming principles.

## Overview

The program allows users to manage restaurant orders by viewing menu items, placing orders, canceling orders, and calculating the total bill including tax and tip suggestions.

This project demonstrates object-oriented design, class structure, and unit testing.

## Features

- View menu items
- Place customer orders
- Cancel orders
- Calculate total bill with tax
- Provide suggested tip amounts
- Filter menu items for heart-healthy options

## Technologies Used

- Java
- Object-Oriented Programming
- Unit Testing

## Purpose

This project was created as part of coursework at Towson University to practice object-oriented programming concepts including class design, data handling, and testing multiple components within a program.
